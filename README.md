# BCCWJ-WLSP-AUX-OT

## Description
BCCWJのPB・PM・PNの一部に分類語彙表番号・助動詞の用法を付与したもの

- BCCWJ-WLSP (内容語の分類語彙表番号)
- BCCWJ-WLSP (れる・られる以外の助動詞の用法付与)
- BCCWJ-AUX (れる・られる　のみ)

の 3つのデータを、BCCWJ-OT (Original Text) のうえで再構成したもの
